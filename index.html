    The Basics

    How this document works, and the vocabulary it uses. Read this before applying anything from the later sections.

    Replace assets

    Imagery and logos load from the local assets/ folders. Replace any file — keeping the same filename — and this page updates automatically. Logos must stay SVG; images are PNG.

    Set the tokens

    Brand name and the four palette colours live in the Customize panel (sidebar, bottom). Changes preview live and are remembered by this browser.

    Edit the text

    Turn on text editing in Customize, then click any paragraph or heading and type. Anything wrapped in [brackets] is placeholder copy waiting for a real brand.

    Terminology

    TermWhat it isUse it for
    HEXSix-digit colour code, e.g. #111111Web, apps, and anything on a screen
    RGBRed / green / blue values, additive light colourScreens, video, digital design tools
    CMYKCyan / magenta / yellow / black ink percentagesPrint production — brochures, packaging, posters
    Pantone (PMS)Standardised spot-ink colours matched by codeBrand-critical print and merchandise where exact colour matters
    LockupA fixed, approved arrangement of logo elementsAny time the logo appears — never rebuild it manually
    WordmarkThe brand name set in its logo formWide formats where the full name should read
    Icon markThe symbol without the nameAvatars, favicons, and very small sizes
    Clear spaceProtected empty area around the logo, measured in xKeeping the mark legible next to other elements
    WeightThe thickness of a typeface style (400, 700…)Building hierarchy without changing typefaces
    Tracking / leadingLetter-spacing / line-height of set typeFine-tuning headlines and body text
    WCAG AAAccessibility contrast standard (4.5:1 for body text)Checking any text-on-colour combination

    Grid

    The proportions the mark is built on. Keep these relationships if you resize or redraw it.

    The mark sits inside a square module; its corner radius and inner circle are sized from that same base unit.

    Clear space & minimum size

    x

    Clear space equals x — the height of the icon mark — on all sides. Nothing enters that zone: no text, no edges, no other logos.

    Digital minimum

    24px height

    Print minimum

    15mm height

    Below minimum size, switch to the icon mark rather than shrinking the full lockup.

    Usage

    Neutral · 60%
    Primary · 30%

    Neutrals carry most of every layout — backgrounds and body text. Primary marks structure and key actions. Secondary is a small, deliberate accent, never a field colour.
